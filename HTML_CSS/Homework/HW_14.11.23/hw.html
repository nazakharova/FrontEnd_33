Task 4 - Block and Inline Elements

Every HTML element has a default display value, depending on what type of element it is. The two most common display values are block (highlighted in blue) and inline (highlighted in yellow).

  1. The <p> tag defines a paragraph.
  2. The <img> tag is used to embed an image in an HTML page.
  3. The <div> element defines a division or a section in an HTML document.
  4. The <pre> tag defines preformatted text.
  5. The <span> tag is an inline container used to mark up a part of a text, or a part of a document.
  6. The <object> tag defines a container for an external resource.
  7. The <form> tag is used to create an HTML form for user input.
  8. The <button> tag defines a clickable button.

Amazon facts

  1. The Amazon is the world's largest tropical rainforest. Covering over 5.5 million km2, it's so big that the UK and Ireland would fit into it 17 times!
  2. The Amazon is found in South America, spanning across Brazil, Bolivia, Peru, Ecuador, Colombia, Venezuela, Guyana, Suriname and French Guiana.
  3. Running through the north of the rainforest is the Amazon River - a network of many hundreds of waterways that stretches 6,840 km.Although there is some debate, most scientists agree that the Amazon is the world's second longest river after the River Nile.
  4. The Amazon has an incredibly rich ecosystem - there are around 40,000 plant species, 1,300 bird species, 3,000 types of fish, 430 mammals and a whopping 2.5 million different insects. Wow!
  5. Around 400-500 indigenous Amerindian tribes call the Amazon rainforest home. It's believed that about fifty of these tribes have never had contact with the outside world!
  6. This area of immense natural beauty plays an important role in limiting climate change. This is because the rich vegetation takes carbon dioxide (a greenhouse gas) out of the air and releases oxygen. You can learn more about this in our feature on the lifecycle of plants!
